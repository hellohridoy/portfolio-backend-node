# Node.js Backend for Portfolio

## Setup
```bash
cd /home/rhossain/Desktop/calculas/portfolio-fullstack/backend-node
npm install
```

## Environment
Default .env is set to Postgres:
- DB: profile
- User: postgres
- Password: postgres

## Run
```bash
npm start
```

API base: `http://localhost:5000/api`

## Key Endpoints
- `GET /api/users/:userId/profile`
- `GET /api/users/:userId/skills`
- `GET /api/users/:userId/programming`
- `GET /api/users/:userId/projects`
- `GET /api/users/:userId/social`
- `GET /api/users/:userId/certificates/primary`
- `GET /api/users/:userId/certificates/secondary`
- `GET /api/users/:userId/education/bsc`
- `GET /api/users/:userId/education/msc`
- `GET /api/users/:userId/cv/download`
